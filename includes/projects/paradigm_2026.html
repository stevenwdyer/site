Paradigm Engineering 2026

Note: This project is currently in development.

Paradigm Engineering is building an autonomous go-kart to compete in the Autonomous Karting Series national grand-prix at Purdue University in May 2026.

The go-kart features a NVIDIA Jetson Orin, a Pixhawk 6C, and an STM32G474 microcontroller along with a suite of other devices including a GPS sensor and antenna, RC receiver, Ubiquiti Bullet AC, and several DC motors. As the software lead, I contribute to all aspects of the custom built software package, which allows for real-time and autonomous control of the go-kart.

Embedded Control Stack (STM32G4)

The STM32G4 is used for real time safety monitoring and motor control. Its firmware is built around FreeRTOS and is organized into defined tasks to complete the following:

  • Receiving control commands from the Jetson over UART via a custom driver.
  • Generating PWM outputs for both steering and braking motors.
  • Interfacing with the drive motor controller via CAN.
  • Monitoring safety conditions such as Jetson heartbeat and e-stop trigger.
  • Diagnostics reporting.

These tasks communicate with one another using RTOS queues. A custom bare-metal UART driver was also written for improved control over timing and error handling.

Autonomous Drive Algorithm

The kart's autonomous drive algorithm is ran on the Jetson Orin and is built using ROS2. The algorithm is given a .txt file with a list of waypoints and a control node iterates through these waypoints, which computes heading and distance errors relative to the kart's current position. Using this information, it will generate steering and velocity commands to guide the kart to the active waypoint. Once reached, the system moves on to the next waypoint.

Computer Vision

The kart's computer vision system utilizes a ZED 2i stereo camera that sends a live camera feed to the Jetson Orin which runs a CNN to detect white lines on the race track. This is used alongside the main autonomous drive algorithm for improved waypoint following.

Custom Flight Software

The kart has custom flight software that runs on the Jetson Orin and is responsible for filtering and fusing sensor data from the Pixhawk. This is still in early development stages.

Simulation

The simulation, based on Gazebo, was developed as part of the 2025 software package. It was used for developing and testing the autonomous drive algorithm with the absence of a physical kart.

Deployment

The embedded firmware is flashed onto the STM32G4 via the SWD protocol using an ST-LINK debugger. The Jetson Orin software is delpoyed using scripted software.
