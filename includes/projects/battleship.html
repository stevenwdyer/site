Battleship

This project is a scaled-down recreation of the strategy game titled "Battleship". It was developed for the ECE 4510 Microprocessors & Digital Logic project at Memorial University of Newfoundland. See this project's repository

This version of the game utilizes two Arduinos that communicate with each other via UART, 3x3 LED matrices for grid visualization, joysticks for ship/shot selection, buttons for ship/shot confirmation, passive buzzers for audio cues, and 7-segment displays for score/state tracking.
